# Module 01 — Mini Challenge: Internet, HTML, CSS

This is a bigger task that combines everything from this module.

Read it carefully before starting. Plan the layout first, then code.

---

## Challenge: Simple responsive landing page

**Goal:** Build a simple landing page for a fictional product or service.

---

## What to build

The page must have these sections, in this order:

### 1. Header / Navigation
- A logo or brand name (text is fine)
- 2–3 navigation links: "Home", "Features", "Contact"
- Links do not need to go anywhere — use `href="#"`

### 2. Hero section
- A big heading (the main message)
- A short subtitle (1–2 sentences)
- A call-to-action button: "Get Started" or "Learn More"

### 3. Features section
- A heading: "Why choose us?" or similar
- 3 feature cards displayed in a row using Flexbox
- Each card has: an icon or emoji, a short title, and 1 sentence of description

### 4. Contact form
- A heading: "Contact Us"
- Name input
- Email input
- Message textarea
- Submit button

### 5. Footer
- A simple line of text: "© 2024 Your Brand Name"

---

## Requirements

- All sections on one HTML file
- CSS in a `<style>` block or separate `style.css` file
- The 3 feature cards must use Flexbox
- The page must be responsive: on small screens (< 600px), cards stack vertically
- Use at least 2 different font sizes
- Use at least 2 different colors

---

## Checklist before finishing

- [ ] Header with navigation links
- [ ] Hero section with heading, subtitle, button
- [ ] 3 feature cards in a row (Flexbox)
- [ ] Contact form with name, email, message, button
- [ ] Footer
- [ ] Cards stack on small screen
- [ ] Page is visually styled (not browser default only)

---

## Tips

- Start with the HTML structure first — no CSS yet
- Add CSS section by section
- Test in browser after each section
- Do not aim for perfect design — aim for working and readable
